# CozeAgent 使用指南布局调整 - 2列布局

## 📋 修改说明

将"使用指南"从自适应布局改为**固定2列布局**（2x2网格）。

---

## 🔧 修改内容

### 修改 app.css

**修改前（自适应）：**
```css
.usage-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(240px, 1fr));
  gap: 20px;
}
```

**修改后（固定2列）：**
```css
.usage-grid {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 20px;
}
```

---

## 📊 布局效果

### 桌面端布局
```
┌──────────────┬──────────────┐
│   🚀         │   ✏️         │
│ 访问智能体    │  输入主题     │
├──────────────┼──────────────┤
│   ✨         │   🎯         │
│ 获取结果      │  应用使用     │
└──────────────┴──────────────┘
```

**固定为 2 列 × 2 行**

### 移动端布局
```
┌──────────────┐
│   🚀         │
│ 访问智能体    │
├──────────────┤
│   ✏️         │
│ 输入主题      │
├──────────────┤
│   ✨         │
│ 获取结果      │
├──────────────┤
│   🎯         │
│ 应用使用      │
└──────────────┘
```

**移动端自动变为 1 列**（响应式设计）

---

## 🆚 布局对比

| 布局方式 | 自适应布局 | 固定2列布局 |
|---------|-----------|------------|
| **桌面端** | 根据宽度自动调整列数 | ✅ 始终2列 |
| **平板端** | 可能3列或2列 | ✅ 固定2列 |
| **手机端** | 1列 | ✅ 1列 |
| **对齐** | 可能不对称 | ✅ 完美对称 |
